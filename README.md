# minimal-alarm

A dead-simple alarm clock that runs entirely in the browser. No frameworks, no build step — just open `index.html`.

## Features

- Live clock display
- Add alarms with an optional label
- Toggle alarms on/off
- Delete alarms
- Beep sound via Web Audio API when an alarm fires
- Alarms persist across page reloads (localStorage)

## Usage

Open `index.html` in any modern browser. No server required.
